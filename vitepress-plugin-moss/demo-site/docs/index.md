# Moss Semantic Search Demo

Welcome to the official documentation for the **Moss Semantic Search** integration. This site serves as a live demonstration of how AI-powered search can improve the user experience of your documentation.

## What is Moss?

Moss is a next-generation search engine designed specifically for technical documentation. Unlike traditional keyword-based search, Moss understands the **meaning** and **intent** behind user queries.

### Key Features

1.  **Semantic Understanding**: Users can find what they need even if they don't use the exact keywords found in the text.
2.  **Zero-Config UI**: Automatically replaces the default VitePress search bar with a premium, responsive modal.
3.  **Local Execution**: Queries run in the browser using WebAssembly, ensuring maximum privacy and lightning-fast results.
4.  **Easy Integration**: Add it to any VitePress site in minutes using the standard plugin system.
